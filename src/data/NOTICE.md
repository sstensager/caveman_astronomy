`stars.json` in this directory is a filtered, reformatted derivative of the
**HYG Database v3** (astronexus/HYG-Database,
https://github.com/astronexus/HYG-Database), which itself compiles data
from the Hipparcos, Yale Bright Star, and Gliese catalogs.

- Source file: `hyg/v3/hyg_v38.csv`
- License: **CC BY-SA 4.0** (https://creativecommons.org/licenses/by-sa/4.0/)
- Filtering: stars with apparent magnitude ≤ 6.5 only (~naked-eye limit);
  the synthetic "Sol" row (id 0) representing the Sun itself is excluded.
- Generated by: `scripts/build-star-catalog.mjs`
