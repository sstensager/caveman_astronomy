# Third-party data notice

`stars.json` in this directory is a filtered, reformatted derivative of the
**HYG Database v3** (astronexus/HYG-Database,
https://github.com/astronexus/HYG-Database), which itself compiles data
from the Hipparcos, Yale Bright Star, and Gliese catalogs.

- Source file: `hyg/v3/hyg_v38.csv`
- License: **CC BY-SA 4.0** (https://creativecommons.org/licenses/by-sa/4.0/)
- Filtering: stars with apparent magnitude ≤ 6.5 only (~naked-eye limit);
  the synthetic "Sol" row (id 0) representing the Sun itself is excluded.
- Generated by: `scripts/build-star-catalog.mjs`

Per CC BY-SA 4.0, this derivative data file is itself distributed under the
same license (share-alike) with the above attribution.

## constellations.json

`constellations.json` is derived from **D3-Celestial**
(ofrohn/d3-celestial, https://github.com/ofrohn/d3-celestial):

- Source files: `data/constellations.lines.json`, `data/constellations.json`
- License: **BSD-3-Clause** (copyright Olaf Frohn)
- Processing: every line-segment endpoint (originally a bare RA/Dec
  coordinate) was resolved to the nearest star already present in this
  app's own `stars.json` catalog (within 1 degree), and only that star's
  Hipparcos id is stored - no coordinates from the source dataset are
  retained in the output. Constellation display names and label anchor
  points (RA/Dec) are carried over as-is.
- Generated by: `scripts/build-constellations.mjs`
